Project

IsIWeb4Shop

A functional online shop demonstrating product navigation, shopping cart management and simple payment. Lightweight MVC architecture in PHP.

Overview

IsIWeb4Shop is a fictional e-commerce shop designed as a portfolio project demonstrating full-stack web development skills. The project showcases a clean MVC architecture, user cart management, order handling and PDF invoice generation.

Key Features

  • Product Catalog: Category-based navigation with filtering and detailed product pages.
  • Shopping Cart: Add items, modify quantities, merge session/user cart.
  • Order Management: Create orders, manage delivery addresses, track status.
  • Authentication: User registration and login with password hashing, user and admin accounts.
  • Reviews & Ratings: Product review system with user ratings.
  • Invoice Generation: PDF export of orders.
  • Payment Handling: PayPal stub and cash payment options.

Technical Architecture

Language & Stack

  • PHP: Framework-less, lightweight MVC architecture.
  • Templating: Twig for views.
  • Database: MySQL.
  • UI: Bootstrap.

Project Structure

  • Controllers: Route handling and request logic.
  • Models: Database access (products, cart, accounts, checkout, etc.).
  • Views: Twig templates.
  • Static assets: CSS, images and resources.

Main Database Tables

  • products, categories, orders, orderitems
  • customers, logins, admin, delivery_addresses, reviews

Learning Outcomes

  • MVC architecture and separation of concerns.
  • User session management and persistent shopping cart.
  • Complete CRUD operations on products and orders.
  • PDF generation and file handling.
  • Authentication system with password hashing.
  • E-commerce integration (cart, payments, address management).
