Neural Networks

A deep dive into self-organizing maps (SOM), parameter analysis and a practical application for a robotic arm.

Overview

This AI project, completed in a pair, focuses on unsupervised learning through Kohonen self-organizing maps. The goal was to approach the topic scientifically by testing, quantifying and visualizing the effect of learning parameters before applying the result to a practical case: predicting the movement of a robotic arm.

Theory and algorithm analysis

A simulation function was used to study how the algorithm behaves on uniform and non-uniform data sets. The analysis focused on three core parameters:

  • Learning rate (η): Ideally around 0.05. Too high a value prevents convergence, while a value of 1 directly overlays neurons on the inputs without global organisation.
  • Neighborhood width (σ): A parameter that controls how densely the map self-organizes. If too low, the neurons do not organize; if too high, the map becomes too concentrated around the center.
  • Time step (N): Set to 30,000 for the final tests. It has limited impact on simple datasets, but a higher N helps stabilize self-organization.

Application: robotic arm

The trained model was then used to solve both forward and inverse kinematics on a two-joint robotic arm.

  • Spatial and motor prediction: By finding the winning neuron, the algorithm can infer the spatial position (x, y) from motor angles (θ1, θ2), and vice versa.
  • Trajectory generation: By mathematically interpolating motor positions between a start and an end point, the map predicts the sequence of spatial positions followed by the robot hand.
