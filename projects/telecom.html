Couverture Réseau

Visualisation interactive de la couverture réseau en France. Une application destinée aux professionnels pour repérer les zones blanches et aux particuliers pour vérifier l'éligibilité de leur logement.

Vue d'ensemble

Ce projet fournit une visualisation interactive de la couverture réseau en France via une interface cartographique et des tableaux de données. Il exploite une base de données répertoriant les communes françaises pour cartographier précisément l'accès au réseau, en offrant des filtres de performance (débit).

Fonctionnalités principales

  • Visualisation interactive : Exploration libre de la carte de France pour analyser la couverture globale.
  • Zoom dynamique : Apparition détaillée des communes lors du zoom sur une zone spécifique.
  • Filtrage par débit : Isolation de la couverture réseau exclusive selon des seuils de débit spécifiques.
  • Mise à jour des données : Structure pensée pour refléter les derniers déploiements du secteur.

Architecture technique

Environnement et Base de données

  • Node.js & npm : Environnement d'exécution et gestionnaire de paquets pour le serveur.
  • Base de données : MySQL, instanciée localement (via XAMPP/Laragon).
  • Données géographiques : Fichier d'initialisation communes.sql regroupant les coordonnées et statistiques.

Structure d'exécution

  • Configuration : Fichier config.json pour la liaison serveur/base de données.
  • Serveur local : Port 3000 utilisé pour le développement et l'accès à l'interface cartographique.

Points clés d'apprentissage

  • Intégration d'une bibliothèque cartographique interactive.
  • Connexion et requêtage entre un serveur Node.js et une base de données MySQL.
  • Traitement et affichage dynamique de données volumineuses (communes françaises).
  • Mise en place de filtres interactifs (débit) modifiant le rendu visuel en temps réel.
