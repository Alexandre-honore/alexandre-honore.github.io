PokeBot Discord

PokeBot is a JavaScript Discord bot built around a clear progression loop: capture Pokemon, run your berry farm, generate income, and reinvest to unlock harder capture areas.

The project combines collection, economy, and power progression. The better your resource management, the more you can target rare captures and stronger Pokemon.

Pitch

A long-term progression bot: capture, harvest, buy, upgrade. Each command feeds a persistent economy that unlocks better capture opportunities.

Concept

The gameplay loop is simple and effective: capture, harvest, buy, improve. The bot turns Discord commands into a persistent progression system focused on resource optimization.

Main mechanics

  • Pokemon capture through dedicated commands.
  • Berry farm generating money in the game economy.
  • Poke Ball purchases to improve capture chances.
  • Unlocking advanced zones with stronger Pokemon.
  • Player profile and progression tracking.

Tech stack

  • Language: JavaScript
  • Platform: Discord bot
  • Hosting: Heroku
  • Database: MongoDB via Mongoose

What this project highlights

  • Designing a readable and engaging progression loop on Discord.
  • Implementing a persistent game economy with MongoDB and Mongoose.
  • Balancing collection, resource management, and difficulty growth.
  • Deploying a continuously available service on Heroku.
